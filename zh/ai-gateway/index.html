AI Gateway

将 Apache APISIX 用作 AI 网关：LLM 代理、负载均衡、token 限流与提示词安全。

ai-proxy

以统一接口代理 OpenAI 兼容的 LLM API。

ai-proxy-multi

跨 LLM 提供商的负载均衡、重试与回退。

ai-rate-limiting

面向 LLM 流量的 token 级限流。

ai-prompt-guard

在请求到达模型前对提示词进行允许/拒绝控制。

ai-rag

在网关层实现检索增强生成。

MCP

了解 Model Context Protocol 与 AI 网关。
