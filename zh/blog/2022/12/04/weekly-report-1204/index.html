从 11.21 - 12.04，有 22 位开发者为 Apache APISIX 提交了 61 个 commits。感谢这些小伙伴为 Apache APISIX 添砖加瓦，是你们的无私付出，让 Apache APISIX 项目变得更好！

导语

Apache APISIX 从开源第一天就以社区方式成长，迅速成为全世界最活跃的开源 API 网关项目。这些成就，离不开社区小伙伴们的共同奋斗。

“独行者速，众行者远”。Apache APISIX 社区周报希望可以帮助社区小伙伴们更好地掌握 Apache APISIX 社区的进展，方便大家参与到 Apache APISIX 社区中来。

我们还整理了一些适合新来社区的小伙伴们参加的 issue！感兴趣的同学们，走过路过不要错过！

贡献者统计

Good first issue

Issue #8413

问题描述：将限制计数插件与支持 tls/ssl 的 Redis Cluster 一起使用。

近期功能特性亮点

Apache APISIX

  • proxy-mirror 插件支持指定镜像请求的前缀（贡献者：Gallardot）
  • jwt-auth 支持隐藏客户端认证凭据（贡献者：pixeldin）
  • APISIX 支持 FIPS 密码套件（贡献者：kingluo）
  • proxy-mirror 插件支持解析域名（贡献者：spacewander）
  • grpc-transcode 插件支持在返回体中展示解析过的 grpc-status-details-bin（贡献者：monkeyDluffy6017）
  • 域名解析时优先解析本地的 /etc/hosts 文件（贡献者：TenYearsIn）
  • proxy-rewrite 插件支持显式设置和删除 header（贡献者：mscb402）
  • 支持国密密码套件（贡献者：spacewander）
  • 支持将敏感字段存储在第三方服务中，如 Vault（贡献者：soulbird kingluo）
  • 支持敏感字段进行自动加密和解密，保护敏感字段（贡献者：tzssangglass）
  • syslog 插件支持设置 log format（贡献者：fatihusta）
  • Admin API 通过 gRPC 协议连接 etcd（贡献者：spacewander）

Apache APISIX Ingress

  • 在日志中隐藏密钥信息（贡献者：macmiranda）
  • 可以通过 annotation 的方式来进行请求方法的控制（贡献者：incubator4）
  • 允许在 APISIX Ingress 与服务注册/发现中心进行集成（贡献者：tao12345666333）

Apache APISIX 的项目官网和 Github 上的 Issue 上已经积累了比较丰富的文档教程和使用经验，如果您遇到问题可以翻阅文档，用关键词在 Issue 中搜索，也可以参与 Issue 上的讨论，提出自己的想法和实践经验。
