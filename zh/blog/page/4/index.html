Apache APISIX 3.7.0 正式发布

我们很高兴地宣布 Apache APISIX 3.7.0 版本已经发布，带来了一系列新功能、Bug 修复和相关用户体验优化。

•
2023年11月21日
Community

社区双周报 (10.23 - 11.05)

最近，我们修复并改进了 Apache APISIX 的部分功能，如支持导出 zipkin 插件的变量和安装依赖项从 apisix-base 改为 apisix-runtime。您可以在本期双周报中详细了解这些新功能。

2023年11月8日
Community

社区双周报 (10.09 - 10.22)

最近，我们修复并改进了 Apache APISIX 的部分功能，如升级 lua-resty-healthcheck 至 3.2.0 版本、支持在 stream 路由里配置 service、authz-keycloak 插件支持使用 secrets 加密，以及 openid-connect 插件支持 authorization_params 参数。您可以在本期双周报中详细了解这些新功能。

2023年10月26日
Community

APISIX Ingress Controller：一种无需 etcd 的轻量级部署方式

APISIX Ingress Controller 创新架构，摆脱对 etcd 集群的依赖，极大简化维护成本和系统复杂性。

•
2023年10月18日
Community

社区双周报 (9.25 - 10.08)

最近，我们修复并改进了 Apache APISIX 的部分功能，如 DNS 服务发现支持指定的 resolv.conf 文件、traffic-split 插件支持 https，以及 Consul 服务发现支持 ACL tokens。您可以在本期双周报中详细了解这些新功能。

Apache APISIX 3.6.0 正式发布

我们很高兴地宣布 Apache APISIX 3.6.0 版本已经发布，带来一系列新功能，修复，和用户体验优化。

•
2023年10月5日
Community
