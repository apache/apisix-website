学生专访｜Apache APISIX 陈焕新——开源不是拦路虎，勇敢迈出第一步

本期开源之夏 2024 学生专访来自 Apache APISIX 社区中选学生陈焕新，他在本届活动中承担的项目是：集成 AWS 和 GCP 的 Secrets Manager。

Community

社区月报 (12.01 - 12.31)

最近，我们新增并改进了 Apache APISIX 的部分功能，包括支持 limit-count 插件使用 plugin metadata 配置速率限制响应头和支持将 system 设置为值并引用 ca_certs 等。有关更多功能新亮点，请阅读本期月报。

2024年12月31日
Community

社区双月报 (10.01 - 11.30)

最近，我们新增并改进了 Apache APISIX 的部分功能，包括新增 ai-content-moderation 和 ai-rag 插件，在 Grafana 仪表盘中新增总请求变化面板等。有关更多功能新亮点，请阅读本期双月报。
